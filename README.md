## ğŸ“± Navigation Flow

The app uses a persistent **Bottom Navigation Bar** for easy access to main sections:

1.  **Notes** (`App Start`): View your list of notes. Tap the **+ FAB** to add a new note. Tap any note to edit.
2.  **Reference**: Browse the scrolling list of cheat sheets and key topics.
3.  **Quiz**: Select a subject from the list to start a quiz. View your past scores in the "History" section below the buttons.
4.  **Goals**: Add new study goals and view your current list.
5.  **Profile**: Check your user details ("Shan e Haider") and study stats.
